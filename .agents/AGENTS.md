# Workspace Customization Rules

## UI/UX Engineering and Quality Standards

Whenever you are tasked with creating, modifying, or refactoring user interfaces, stylesheets, components, layouts, or page structures in this workspace:

1. **Read the Playbook**: You MUST read the [UI Playbook.md](file:///c:/Users/karim/Desktop/afaq-alquran/UI%20Playbook.md) to align on the core design philosophy, responsive engineering principles, and quality checklists.
2. **Strict Adherence**: Apply all instructions regarding responsive-first layout, spacing, accessibility (a11y), visual hierarchy, and component design described in the playbook.
3. **Quality Check**: Before completing any UI task, verify your changes against the "Enterprise Production" quality standard defined in the playbook.
